# CMD命令

| 命令               | 描述                    |
| ------------------ | ----------------------- |
| dir . /B > a.csv   | 遍历文件夹文件名到a.csv |
| rename 1.txt 2.txt | 修改文件名              |

# Excel函数

| 函数                             | 结果                                |

| TEXT(A1,"000")                   | 001，数字前填充0                    |
| RIGHT(A1，Len(A1)-Find("-"，A1)) | 字符串找到第一个"-"截取以后当字符串 |
